Release Notes

0.5 (2016-07-07)

This update includes the following changes:

  • Use xercesImpl-xsd11-shaded to avoid conflicts with components that depend on Xerces 2.11.0.
  • Add support for command-line arguments (org.opengis.cite.kml2.CommandLineArguments)
  • Update parent POM (ets-common-7)

0.4 (2016-02-22)

First publicly available release. This test suite may be used to validate any OGC KML 2.x representation; it supersedes the previous KML 2.2 test suite.

  • Fix issue #1: Tile parameters not replaced when checking PhotoOverlay icon
  • Update references to final OGC KML 2.3 specification
  • Update dependencies (parent POM)

0.3 (2015-07-23)

This maintenance release includes several refactorings and improved unit test coverage.

  • Reorganize feature tests (CL1)
  • Use JAX-RS Client in LinkValidator
  • Fix Javadoc errors when building with JDK 8
  • Add unit tests

0.2 (2015-07-10)

This update includes tests covering all conformance levels (CL1, CL2, CL3) in accord with the specifications formally adopted by the OGC.

0.1 (2015-05-01)

This initial release runs the following tests against a KML 2.x document:

  • ATC-102: Schema validity
  • ATC-103: Valid geometry coordinates
